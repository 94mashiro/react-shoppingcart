# AppCart
**一款可自行添加 AppStore 应用的价格监控 WebAPP。**
## 技术栈
**前端**

* react
* react-router
* redux
* react-redux

**后端**

* flask
* beautifulsoup
* peewee
* apscheduler
* requests
